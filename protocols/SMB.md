# Server Message Block (SMB) protocol
This protocol allows a server to share files with remote computers.

Eg. We have a laptop, and a printer that is connect to the company's PC. To print something on our laptop, our laptop sends a print request to the company's PC via the SMB protocol. Through the same route, your laptop receives a response on whether the file is being printed or not.<br>

Current version is SMB V3.<br>

### Fun fact
The infamous Wannacry ransomware used a vulnerability in the SMB protocol V1 to infect thousands of computers.
